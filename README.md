# Variant Group Test

## Структура

- `src/kanban` – компоненты фичи
- `src/ui` - дизайн-система + UI Kit; ключевые файлы:
  - `src/ui/styles.css` – токены, базовые и глобальные стили
  - `src/ui/UIText.tsx` – компонент для стилей текста. "Стиль текста" в данном случае это комбинация font-size/font-weight/line-height/letter-spacing, таких комбинаций в приложении ограниченное количество. Каждый раз "собирать" их из токенов неудобно, лучше использовать готовый компонент
  - `src/ui/layout.tsx` – вспомогательные лейаут-компоненты

### Про общие компоненты:
- Из представленного макета выделил только `Avatar` и `Tag` – вероятно они могут быть переиспользованы в других местах.
- Делать компонент `Card` посчитал пока излишним, т.к. другого использования карточек нет. Возможно представленный вид уникален именно для доски.
- Заголовок колонки с моей точки зрения тоже не является самостоятельным компонентом, это часть компонента колонки (Lane в фигме). Объединять его, например, с тегом в какой-то более сложный компонент `Badge` с вариантими не стоит.
